# Track 1 — 일간 다이제스트 템플릿

> **목적**: 본인이 매일 5~15분 안에 *오늘 봐야 할 것*만 인지.
> **생성 시점**: 매일 한국 시간 오전 7시 (미국 시장 마감 후) 또는 본인 호출 시.
> **검토 시간**: 10~15분
> **사용 방법**: 본인이 claude.ai 또는 Claude Code 에서 "오늘 일간 다이제스트 만들어줘" 또는 Phase γ 도달 시 자동 푸시.

---

## [일간 다이제스트 — YYYY-MM-DD]

### 0. 한 줄 헤더
> 오늘의 핵심 1줄. P1 이벤트 있으면 그것, 없으면 시장 전체 큰 흐름.

---

### 1. 🚨 P1 이벤트 (즉시 의사결정 필요)
> 본인이 *오늘 안에 결정해야 할 것*. 없으면 "해당 없음".

각 항목 형식:
- **이벤트 명**
- 영역·종목 영향
- 본인이 검토할 것 (액션 아이템)

예시:
- **GE Vernova 가이던스 +12% 상향**
  - 영역: AI DC 전력 인프라 테제 강화. backlog $163B 도달.
  - 액션: GEV 현재 보유 비중 점검. 추가 매수 검토.

---

### 2. 📈 워치리스트 가격 변동 (±3% 이상만)
> 노이즈 제거. 큰 변동만.

| 티커 | 기업명 | 전일 종가 | 변동 | 사유 (있는 경우) |
|------|-------|---------|------|--------------|
| 종목1 | | | +X.X% | 분기 실적 leak |
| 종목2 | | | -X.X% | 한국 시장 변동 |

### 3. 🎯 영역별 변화 (Mega Change Map)
> Tier 1·2 영역들의 그날 동향. 변화 없는 영역은 생략.

#### Tier 1 영역
- **영역명**: 변화 한 줄. (예: "AI DC 전력 — 빅테크 4사 Q1 가이던스 일제 상향")
- **영역명**: ...

#### Tier 2 영역
- **영역명**: 변화 한 줄.

#### Mega Map 자체 변화
- 점수 변동 있는 영역: (예: "원자력 르네상스 시차 +3 → +4 상향")
- Tier 이동 후보: (있으면)

### 4. 📰 주요 기업 동향 (워치리스트 28종목 + 인접)
> 가격 변동 없어도 *중요 뉴스* 있는 종목.

각 항목 1~2줄:
- **티커 (기업명)**: 동향 1줄
  - 예: **PLTR (Palantir)**: 정부 신규 계약 $200M 발표. Defense vertical 매출 모멘텀 유지.
  - 예: **128940 (한미약품)**: efpeglenatide 식약처 추가 자료 제출 완료. 허가 일정 영향 모니터링.

## 사용 가이드

### 본인이 매일 할 것
1. **5분 검토**: 섹션 0~4 까지. P1 이벤트·가격 변동·기업 동향만 빠르게.
2. **10분 검토** (시간 있을 때): 섹션 5~7 추가. 다음 주 준비 + 본인 강조 종목.
3. **15분 검토** (주말 전·시간 충분): 전체. 섹션 6 의 제안 사항 검토.

### 본인이 안 볼 것
- 다이제스트에 포함되지 않은 모든 것 (노이즈)
- 변동 없는 영역·종목
- Claude 가 P3 이하로 분류한 이벤트 (별도 요청 시에만)

### 트리거 룰 (어느 정보가 다이제스트에 올라가나)

**섹션 1 (P1 이벤트)**:
- 워치리스트 종목 가이던스 ±10% 이상 변경
- 영역 핵심 정책 변화 (트럼프 행정명령, EU AI Act 등)
- 워치리스트 종목 M&A 발표
- 빅테크 분기 실적 (CapEx 가이던스 변경)

**섹션 2 (가격 변동)**:
- 절대 변동 ±3% 이상만
- (작은 변동은 노이즈)

**섹션 3 (영역 변화)**:
- A/B/C 시차 점수 변경
- 잣대 점수 변경
- Tier 강등·승급 시그널

**섹션 4 (기업 동향)**:
- 워치리스트 28종목 + 직접 peer 한정
- 분기 실적·가이던스·M&A·대형 계약
- 무관한 일반 뉴스 제외

**섹션 5 (이벤트)**:
- 향후 7일 P1·P2 이벤트만
- P3 은 주간 다이제스트로

**섹션 6 (제안)**:
- 최대 2개. 더 많이는 노이즈.

**섹션 7 (Track 4)**:
- 본인이 *명시적으로 추가* 한 종목만
- "{종목} 강하게 본다, Track 4 추가" 같은 명시 요청 필요
